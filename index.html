2024 Presidential Election Prediction

What are the features that affect the US citizens vote?
Machine learning helps to predict the US vote.

We worked with different features to understand which ones weighed more for predicting the election votes by county. The predictive model is based on a Supervised Machine Learning model called Random Forest. The model was selected because of its power to find relationships and its ease of interpretation. We got an accuracy score of 87% with this model, giving us a hint that the features used have high importance in predicting the winner party (Democrat versus non Democrat) In order to improve this number it it might be necessary to add more data and features.
